# Keyboard shortcuts

## Main area

|Sequence|Description|
|---|---|
|`n`|Open Post dialog|
|`s`|Open Search dialog|
|`c`|Open Add Column dialog|
|`1`～`9`|Display columns according to number|
|`Ctrl`+`←`|Display first column|
|`Ctrl`+`→`|Display last column|

## Post/Search dialog

|Sequence|Description|
|---|---|
|`Ctrl`+`Enter`|Post/Search|
